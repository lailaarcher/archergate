The problem

Licensing infrastructure is a full-time job you didn't sign up for.

Key generation. Machine validation. Offline handling. Expiry logic. Server management. Every hour you spend on this is an hour not spent on your product. Enterprise solutions exist, but they cost enterprise money and lock you into someone else's platform.

What we did about it

One library. Thirty minutes. Full ownership.

Archergate is a licensing SDK that links directly into your binary. Machine-bound keys. 30-day offline validation. 14-day trials. Cryptographic integrity checks. MIT licensed. Self-hostable. You own the code, the server, and every decision about how your licensing works.

Capabilities

What's inside

Everything you need to ship professional licensing. Nothing you don't.

Machine binding

Hardware-locked keys

SHA-256 fingerprint derived from CPU identity and OS install ID. Each license is bound to the machine it was activated on.

Offline validation

30-day grace period

One successful validation caches the license locally for a month. Touring producers, remote studios, unreliable connections — all handled.

Trial system

14-day evaluation

Built into the SDK. No server calls needed, no user signup, no tracking. Users try it, then decide.

Integrity

Cryptographic verification

HMAC-signed cache files. Validation receipts with timestamps. Atomic heartbeat counter. Multiple independent verification paths.

Languages

Rust, C, C++

Native Rust SDK with C FFI and C++ RAII wrapper. Works in any native binary -- JUCE plugins, Unity native code, Tauri apps, game engines, or standalone tools.

Infrastructure

Self-host or use ours

Reference server ships with the SDK. Axum + SQLite. Deploy it yourself, modify it, or use Archergate's hosted option.

Common questions

Is this free?

The SDK is MIT licensed and costs nothing. We may offer paid managed hosting and support later, but the core tooling stays free.

Can I run my own server?

Yes. A reference server (Rust + SQLite) ships with the SDK. Deploy it wherever you want, modify it however you need.

What data gets collected?

Self-hosted: nothing leaves your server. Archergate cloud: license key, machine fingerprint, activation date. No analytics, no tracking, no resale.

What about offline users?

After one online validation, the license works offline for 30 days. HMAC-signed local cache. No internet required during that window.

Do I ship a separate DLL?

No. Static library links directly into your binary. One file, no runtime dependencies, clean distribution.

What if Archergate goes away?

The SDK is open source. Self-host, fork, or replace the server. Nothing about your licensing depends on us existing.
