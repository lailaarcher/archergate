**PR title:** Add tauri-plugin-archergate (software licensing plugin)

**PR body:**

Tauri plugin for adding machine-bound licensing to desktop apps. Validates license keys against a self-hosted server and locks them to the user's hardware fingerprint. Works offline for 30 days after initial validation. Includes TypeScript bindings for the frontend (React, Svelte, Vue).

- MIT licensed
- Rust backend with Tauri v2 plugin API
- TypeScript frontend bindings included
